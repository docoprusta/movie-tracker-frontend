Welcome to movie tracker!

What is this app?

You can use this app for tracking your movies, series and other video files played with vlc player.

You can see the close date, the title, the season and the episode numbers (if it is an episode of a series), the last position of the file and the files' duration.

How to use this app?

  1. If you are not registered Register, then Login

  2. Create a config.txt file with the following content:

    http://movie-tracker-backend.herokuapp.com/videos
{{authService.getApiKey()}}
  3. Download the Lua extension for vlc

  4. Choose an os:

    Windows

    1. Install curl

      Download the installer from here and install it

    2. Paste config.txt and 0Awn22Xs.lua to

      For all users 

      <VLC_INSTALLATION_FOLDER>\lua\extensions

      example 

      C:\Program Files\VideoLAN\VLC\lua\extensions

      For current user 

      %APPDATA%\vlc\lua\extensions\

    Linux

    1. Install curl

      A few linux distros are shipped with curl so check it with 

      curl --version

      Install for debian based distros

      sudo apt install -y curl

      or

      sudo apt-get install -y curl

      If you use a different linux distro check it on the package manager

      Alternatively you can find the binary here

    2. Paste config.txt and 0Awn22Xs.lua to

      For all users 

      /usr/lib/vlc/lua/playlist/

      or 

      /usr/share/vlc/lua/extensions/

      For current user 

      ~/.local/share/vlc/lua/extensions/

    MacOS

    I have not tested it in MacOs!

    1. Install curl

      Installation guide

    2. For all users 

      /Applications/VLC.app/Contents/MacOS/share/lua/extensions/

      For current user 

      /Users/%your_name%/Library/Application Support/org.videolan.vlc/lua/extensions/

  5. Open a video file with vlc

  6. If you want to send data to the server and quit:

    VLC Menu --> View --> Send data to the server and quit

    After that you can check your videos if you the videos menu on the navbar
